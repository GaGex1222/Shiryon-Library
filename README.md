### 📊 ניהול מלאי דינמי
המערכת מאפשרת שליטה מלאה על מצבת הספרות בכל רגע נתון:
* **הוספת ספרים:** הזנת כותרים חדשים למערכת ושיוכם לאחד מ-9 הארונות.
* **עדכון כמויות:** ממשק מהיר להוספה או גרעון של עותקים (Inventory Management) בזמן אמת.
* **מעקב סריקה:** ניהול רישומי השאלה והחזרה בצורה דיגיטלית.

### 📥 מערכת בקשות למשיכת ספרות
אחד הפיצ'רים המרכזיים הוא טופס בקשה מובנה:
* משתמשים (קצינים/חיילים) יכולים להגיש **בקשה דיגיטלית** למשיכת ספרות ספציפית.
* המערכת בודקת זמינות מלאי ומפיקה אישור למשיכה.
* היסטוריית בקשות נשמרת לצורכי בקרה של מפקדת קצין שריון ראשי.

## 🛠️ טכנולוגיות בשימוש
הפרויקט נבנה כחלק ממערך הדיגיטציה של החיל, תוך שימוש ב-Tech Stack מודרני:
* **Frontend:** React / Next.js (ממשק משתמש מהיר ונוח).
* **Backend:** Node.js (ניהול הלוגיקה וה-API של הספרייה).
* **Database:** SQL / PostgreSQL (ניהול מסדי נתונים של אלפי ספרים וחוברות).
